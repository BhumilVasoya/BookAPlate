## Description
**BookAPlate** is a full-stack restaurant table reservation web application that allows users to easily book tables at restaurants.  
The system helps restaurants manage reservations efficiently and provides customers with a smooth booking experience.

The frontend is built using **React.js**, while the backend is developed using **Spring Boot**, following a RESTful architecture.

---

## Features
- User-friendly table reservation system
- View available tables and time slots
- Secure backend APIs
- Responsive user interface

---

## Tech Stack

### Frontend
- React.js
- JavaScript (ES6+)
- CSS3
- Axios

### Backend
- Spring Boot
- Java
- Spring Web
- Maven

### Database
- MySQL

## How to Run

### Backend
1. Open backend folder
2. Run: mvn spring-boot:run


### Frontend
1. Open frontend folder
2. npm install
3. npm start
